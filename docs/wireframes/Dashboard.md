# Dashboard Wireframe

Purpose: rychlý přehled výkonnosti sales týmu a přehled leadů.

Sections:

- Header
  - Logo, název, jazyková volba, témata, uživatelské menu

- KPI strip (3 karty):
  - Firmy (počet)
  - Průměrné skóre leadů
  - Top lead (jméno)

- Hlavní obsah (dvou-sloupcový):
  - Levý: seznam leadů (card list) s tlačítkem "Otevřít" pro detail
  - Pravý: aktivity / feed nebo grafy (zatím placeholder)

- Dolní oblast: Automations / Integrace panel

Notes:
- Priority 1: čitelnost, kontrast, rychlé akce (open, create opportunity, send email)
- Priority 2: responsivita, modal pro detail leadu s ARIA atributy

Colors / spacing: použít Tailwind utility, větší padding na kartách, jemný stín pro elevaci.
